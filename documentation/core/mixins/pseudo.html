Pseudo Elements

Mixins for pseudo elements.

pseudo

$side: can be 'before', 'after', or 'both' 
@mixin pseudo($side: "before") {
  position: relative;
  @if $side == "both" {
    &:before, &:after {
      position: absolute;
    }
  }
  @else {
    &: {
      position: absolute;
    }
  }
}
