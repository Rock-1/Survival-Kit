Drupal Theme

Reset

Choose a CSS reset.

reset init

  • pick a reset or normalization
  • accepted values: ‘meyer’ (default), ‘normalize’, ‘none’
  • choose to reset header sizes to 1em (applies to ‘normalize’ only)
  • Uses Eric Meyer reset 2.x and Normalize
  • use Formalize
$init-type: meyer !default;
$init-headers: true !default;
@mixin init($init-type, $init-headers) {
  @if $init-type == meyer {
    @include meyer-reset;
  }
  @if $init-type == normalize {
    @include normalize;
    @if $init-headers {
      // reset the header sizes
      h1, h2, h3, h4, h5, h6 {
        font-size: 100%;
        margin-top: 0;
        margin-bottom: 0;
      }
    }
  }
}
